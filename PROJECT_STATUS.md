---

## 🎯 项目概况

**产品定位**：为品牌部门举办的论坛、沙龙类活动提供一体化的轻量报名管理工具。

**技术栈**：
- 后端：FastAPI + SQLAlchemy + MySQL
- 管理端：Vite + React + TypeScript + Ant Design
- 小程序端：Taro + React + TypeScript

**核心用户**：
- C端：参会老师（活动浏览、报名、签到、互动）
- B端：品牌部门管理员（活动发布、报名审核、数据分析）

## 🔧 关于审核功能的架构决策

**决策：保留但可配置控制**

- 通过 `activity.requires_approval` 字段控制
- 默认 `false`（无需审核，自动通过）
- 批量审核接口保留，仅在需要时调用
- 前端UI根据配置动态显示/隐藏
