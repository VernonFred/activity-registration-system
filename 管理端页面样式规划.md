管理端页面样式规划
左侧导航栏：
1、仪表盘：有3张大卡片
    （1）活动卡片
        - 卡片左边：展示活动的标题、简介、状态（报名状态之类的）
        - 卡片右边：小标题（活动数据）、报名人数、通过人数、报名用户的圆形小头像
    （2）最近通知卡片
        - 关于通知提醒的消息
    （3）活动流卡片
        - 用户圆形头像/姓名
        - 用户行为：刚刚点赞了某个活动、刚刚收藏了某个活动、刚刚评论了某个活动等等
2、活动创建
    （1）右上角按钮「+ 新活动」（固定在 PageHeader 右侧 extra）
        - 弹窗表单三段：
            ① 基本信息（胶囊标题）：
               - 标题
               - 标题下方三列并行：封面图（上传）/ 活动群二维码（上传）/ 缴费二维码（上传，受“是否缴费”开关控制）
               - 是否缴费（胶囊标题+开关）：下方仅“缴费链接”输入框（关闭后禁用，同时禁用“缴费二维码上传”）
               - 标签、简介
               - 地区（省/市/区级联 → 然后根据所选地区提供热门酒店联想，亦可自定义输入；不再单独设置“城市”字段）
               - 详情内容（富文本编辑，工具栏：字号/字体/颜色/列表/缩进/引用/代码块/图片/PDF 链接）
            ② 时间设置（胶囊标题）：报名时间 / 签到时间 / 会议时间 使用 3 个 RangePicker 展示（开始/结束成对选择）。
            ③ 发布与审核（胶囊标题）：状态（草稿/即将开始/报名中/已结束/已归档）、需要审核。
        - 数据映射：
            - 封面图、活动群二维码、缴费二维码 → `extra.cover_image_data` / `extra.group_qr_data` / `extra.payment_qr_data`
            - 缴费说明/链接 → `extra.payment_note`
            - 地区路径 → `extra.region_path`（数组 [省, 市, 区]），若选择了地区自动回填 `city=市`（可手动覆盖）
        - 富文本：React Quill，工具栏含“图片”；图片以 dataURL 直接插入，PDF 插入“文件名 (PDF)”超链接
    （2）顶部筛选：简洁行内布局（Segmented + 搜索），不再使用长条卡片背景
    （3）活动列表使用 SectionCard 容器；列：标题/城市/状态（中文胶囊）/创建时间/操作【编辑（打开同样样式弹窗）/删除（确认框，胶囊按钮）】；启用表格分页；顶部提供地区 Chips（按省），并提供“批量删除/清除选择”胶囊按钮（常显，随选择禁用/启用）
3、报名管理
    （1）可以筛选活动，然后下方就是关于用户填写的报名信息的所有内容（接站点、房型偏好等这些信息）/签到，用表格形式呈现。
    （2）动态表头：根据所选活动的表单字段实时生成列。
    （3）筛选：提供“报名状态/签到状态”的 Segmented 筛选；支持导出报名 CSV。
4、用户管理
    用户信息
        - 用户列表
            （1）用户的个人信息
                - 头像、姓名、学校和部门/职位
                - 主要信息：学校、学院/部门、职位
                - 联系方式：手机号码、电子邮件
                （2）用户行为
                    - 参与活动
                        - 已参加活动列表（包含该活动的点赞、评论、分享、收藏）
                    - 最近评论
                        - 对参加活动的评分与评论



5、审核管理
    （1）审核列表（批量审核、批量删除、批量导出）
        - 表头：姓名、学校、手机号、是否审核、是否缴费、缴费截图
        - 流程：人工审核、自动审核
        - 通知：已通知、未通知

5、审核管理
    （1）审核列表（批量审核、批量删除、批量导出）
        - 表头：姓名、学校、手机号、是否审核、是否缴费、缴费截图
        - 流程：人工审核、自动审核
        - 通知：已通知、未通知

6、通知管理
    （1）通知提醒
        - 通知设置
            - 模板样式
            - 通知编辑（有几个固定的场景通知，需要事件触发自动发送）
                - 编辑：标题、类型、内容、时间
                - 
                    - 场景一：报名审核中
                    - 场景二：报名审核通过
                    - 场景三：报名成功
                    - 场景四：还未缴费
                    - 场景五：缴费成功（这个需要用户上传了缴费截图之后，自动触发）
                    - 场景六：活动结束
                    - 场景七：徽章发放
                    - 场景八：邀请评价/问卷调查
5、活动数据：
    （1）顶部：左侧是下拉框的活动筛选
    （2）顶部下方：报名总学校数、报名总数、签到总数（3张卡片）
    （3）中间图表区域：前20名学校报名人数对比
    （4）图表区域下方：【报名学校人数】卡片
        - 左卡片：收藏和分享卡片
        - 中间：星级评分图表
        - 右卡片：点赞和评论卡片
7、徽章管理
    （1）徽章概览
        -全部徽章
            - 启程系列
            - 互动系列
            - 连续参与系列
            - 荣誉成就系列
            - 隐藏菜单系列
    （2）创建徽章
        - 左侧
            - 基本信息
                - 徽章名称
                - 所属系列
                - 徽章描述
                - 获取提示语
            - 徽章外观
                - 徽章颜色
                - 徽章图标
        - 右侧
            - 徽章预览
    （3）徽章规则
        - 获取条件设计（拖拽下方条件模块到右侧区域，组合设计徽章获取规则）
        - 徽章规则管理（导出规则、新建规则）
            - 规则名称
            - 触发条件
            - 关联徽章
            - 状态
            - 操作
                - 编辑
                - 删除
    
    暂定的徽章系列
            启程系列徽章
                            ├─ 初次登场 / 成功入选 / 准时到场 / 全勤达人 
                            初次登场
                                规则条件：- 首次报名活动
                                获取提示语：- “你的第一场活动，正式启程！”
                            成功入选；
                                规则条件：- 报名成功
                                获取提示语：- “恭喜加入活动名单“
                            准时到场；
                                规则条件：- 第一次签到成功
                                获取提示语：- “守时是你的闪光点！”
                            全勤达人；
                                规则条件：- 报名的所有活动均签到成功
                                获取提示语：- “全勤出席，完美表现！”
                        互动系列徽章
                            ├─ 开口有料 / 金句制造机 / 人气发言者/任务执行者/连续打卡/活力不息
                            开口有料
                                规则条件：- 首次评分或发布评论
                                获取提示语：- “第一次互动，就很精彩！”
                            金句制造机
                                规则条件：- 首次发布长评论（超过 100 字）或者评论被官方精选
                                获取提示语：- “你的观点，被全场记住！”
                            人气发言者
                                规则条件：- 评论点赞或回复数进入前10%
                                获取提示语：- “讨论因为你而更热烈！”
                            任务执行者
                                规则条件：- 完成活动测评或调查问卷
                                获取提示语：- “任务完成，认真态度值得肯定！”
                            连续打卡
                                规则条件：- 连续参与2场及以上活动
                                获取提示语：- “保持热度，精彩延续！”
                            活力不息
                                规则条件：- 连续参与4场活动
                                获取提示语：- “活力常在，你是活动的常青树！”
                        荣誉成就类
                            ├─ 徽章收藏家 / 活动助力官 / 活动之星
                            徽章收藏家
                                规则条件：- 获得3枚徽章
                                获取提示语：- “恭喜成为小小收藏家”
                            活动助力官
                                规则条件：- 主动帮助他人报名
                                获取提示语：- “你的热心，让活动更温暖！”
                            活动之星
                                规则条件：- 获得10枚徽章
                                获取提示语：- “徽章墙上的闪光记忆属于你！”
                        隐藏彩蛋类
                            ├─ 闪电报名王 / 午夜打卡者 / 周年纪念章/ 沉默观察员 
                            闪电报名王
                                规则条件：- 报名用时小于5分钟
                                获取提示语：- “手速无敌，报名第一！”
                            午夜打卡者
                                规则条件：- 晚上10点之后报名
                                获取提示语：- “终于想起报名了”
                            沉默观察员
                                规则条件：- 参与2次活动但未发布评论
                                获取提示语：- “沉默不语，但你始终在场。”
                            周年纪念章
                                规则条件：- 注册满1年自动解锁
                                获取提示语：- “感谢一路同行，一周年快乐！”                            
8、评论管理
    （1）筛选条件
        - 活动名称
        - 评分等级
            - 5星
            - 4星
            - 3星
            - 2星
            - 1星
        - 评论状态
            - 全部状态
            - 普通评论
            - 置顶评论
            - 精选评论
            - 已隐藏
        - 评论时间
        - 关键词搜索
    （2）评论列表
        - 全选、批量精选、批量删除
        - 评论列表
            - 评论内容
            - 精选、置顶、删除
    


9、表单设计

10、设置管理
    （1）传统的设置管理（包括密码设置等等）
## 进度同步
- 操作区按钮统一使用胶囊风样式：`pill-action` 与 `pill-delete`，小尺寸 `size="small"`。
- 报名管理页的“批量导出 Excel”“批量删除”“清除选择”样式与“活动创建”页一致，显示选中计数。
- 删除操作统一使用 `Popconfirm` 二次确认，保持交互一致性。
- 搜索与筛选采用胶囊输入组件，重置按钮对齐。
- 表单设计页：采用拖拽式生成器（`fr-generator`），左侧字段库 + 中间编辑区 + 右侧属性面板布局，对齐腾讯文档风格；保存后实时更新活动的 `form_fields`。
