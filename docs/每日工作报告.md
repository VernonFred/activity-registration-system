#### 5. 修复暗黑模式时间图标颜色 ✅

**完成时间**: 10:10 - 10:15  
**任务编号**: 图标颜色修复  
**用时**: 5 分钟

**问题**: 时间徽章前的日历图标是黑色，深色背景上看不清

**修复方案**:
```scss
// 浅色主题
.time-icon {
  filter: brightness(0) saturate(100%) invert(45%) 
          sepia(95%) saturate(1500%) hue-rotate(5deg);
}

// 深色主题
.time-icon {
  filter: brightness(0) saturate(100%) invert(75%) 
          sepia(50%) saturate(1000%) hue-rotate(360deg);
  opacity: 1;
}
```
