## 📋 归档规则

### 时间规则
- **归档周期**: 每周（周一至周日）
- **归档时机**: 每周日晚或下周一早
- **文件命名**: `YYYY-MM-WW_起始日.结束日.md`
  - 示例：`2025-12-W2_12.08-12.12.md`

### 归档内容
归档文档包含该周的：
- ✅ 所有已解决的问题
- ✅ 所有已关闭的问题
- ✅ 该周的讨论和决策记录
- ✅ 完整的问题状态变更历史

### 当前文档

- 仅保留**本周进行中**的问题

## 🔍 如何查找历史问题

### 按日期查找
1. 确定问题提出的日期
2. 找到对应周次的归档文件
3. 在文件中搜索关键词

### 按关键词查找
```bash
# 在所有归档中搜索
grep -r "关键词" docs/问题跟踪归档/

# 在特定归档中搜索
grep "关键词" docs/问题跟踪归档/2025-12-W2_12.08-12.12.md
```
