## 🔧 安装命令

### 小程序端

```bash
# 进入项目目录
cd frontend/app

# 安装依赖
pnpm install

# 如果需要安装特定依赖
pnpm add <package-name>

# 如果需要安装开发依赖
pnpm add -D <package-name>
```

### 后端

```bash
# 进入项目目录
cd backend

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
source venv/bin/activate  # macOS/Linux
# 或
venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt
```

## ⚠️ 注意事项

### 版本兼容性

1. **Taro 版本一致性**: 所有 `@tarojs/*` 包必须使用相同版本（当前为 3.6.26）
2. **React 版本**: 确保 React 版本与 Taro 兼容

### 已移除的依赖

| 依赖名称 | 移除原因 | 移除时间 |
|----------|----------|----------|
| tdesign-icons-view | 小程序端不兼容 | 2025年12月02日 |

### 依赖安装流程

1. **告知用户**: 需要安装新依赖时，先在对话中告知用户
2. **提供命令**: 提供完整的安装命令
3. **用户执行**: 由用户在终端执行安装命令
4. **更新文档**: 安装完成后更新本文档
