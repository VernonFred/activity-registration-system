## 📱 活动详情页设计规范

### 页面结构

```
┌─────────────────────────────────┐
│  顶部导航（半透明）               │
│  ← 返回    活动标题              │
├─────────────────────────────────┤
│  大图封面（沉浸式，带渐变蒙层）    │
├─────────────────────────────────┤
│  四Tab导航栏                     │
│  速览 | 议程 | 酒店 | 直播       │
├─────────────────────────────────┤
│                                 │
│  内容区域（根据Tab动态切换）      │
│                                 │
├─────────────────────────────────┤
│  底部胶囊浮岛（固定）             │
│  ⭐💬❤️↗️    [立即报名]         │
└─────────────────────────────────┘
```

### 底部胶囊浮岛

```scss
.floating-action-bar {
  position: fixed;
  bottom: 32rpx;
  left: 32rpx;
  right: 32rpx;
  height: 120rpx;
  background: #1A1A1A;
  border-radius: 48rpx;
  box-shadow: 0 -8rpx 32rpx rgba(0, 0, 0, 0.3);
  display: flex;
  align-items: center;
  padding: 0 32rpx;
  z-index: 1000;
}
```

**按钮布局**:
- 左侧：4个图标按钮（收藏、评论、点赞、分享）
- 右侧：立即报名按钮（宽度 240rpx，橙色背景）

### 四Tab导航

| Tab | 内容 |
|-----|------|
| 速览 | 封面、标题、时间、地点、报名人数、活动介绍 |
| 议程 | 时间轴、议程卡片 |
| 酒店 | 酒店选择、房型价格、位置地图、交通指南、天气 |
| 直播 | 大图展示、查看直播按钮 |

### 开发优先级

| 优先级 | 内容 | 原因 |
|--------|------|------|
| **P0** | 速览Tab + 底部浮岛 | 核心功能 |
| **P0** | 议程Tab | 核心功能 |
| **P1** | 评论系统 | 重要社交功能 |
| **P1** | 酒店Tab | 辅助功能 |
| **P2** | 直播Tab | 简单，后续完善 |

### 需要的新组件

| 组件名 | 用途 | 复杂度 |
|--------|------|--------|
| `TabBar` | 四Tab切换 | 简单 |
| `Timeline` | 时间轴（议程） | 中等 |
| `Rating` | 星级评分 | 简单 |
| `CommentCard` | 评论卡片 | 中等 |
| `HotelCard` | 酒店信息卡片 | 复杂 |
| `WeatherCard` | 天气信息 | 简单 |
| `FloatingActionBar` | 胶囊浮岛 | 中等 |
