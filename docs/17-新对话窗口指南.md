## ⚠️ 重要提醒

### 1. 上下文限制

当对话过长时，AI 可能会忘记之前的内容。建议：
- 单次对话不超过 **50-100 轮**
- 完成一个功能模块后开启新对话
- 重要决策记录到文档中

### 2. 文档同步

每次对话结束前，确保：
- 进度更新到 `1-项目进度文档.md`
- 新任务添加到 `16-下一步规划文档.md`
- 问题记录到 `8-问题跟踪文档.md`
- 代码提交到 GitHub

### 3. Git 提交

每次重要修改后：
```bash
git add .
git commit -m "描述修改内容"
git push
```
